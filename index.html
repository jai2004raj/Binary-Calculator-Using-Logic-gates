1. Purpose & Overview

This simulator models a 4-Bit Parallel Adder-Subtractor. It demonstrates how digital logic gates perform arithmetic by manipulating binary voltage levels (High/Low).

2. Subtraction via 2's Complement

The circuit utilizes four Full Adders. Subtraction is performed by XORing input B with the Mode bit. When Mode=1, B is inverted (1's complement) and the initial carry-in adds 1, completing the 2's complement process.
